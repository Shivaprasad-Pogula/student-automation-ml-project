# Simple Automation + ML Project

This is a basic student project for Cognizant skill cluster selection.

## What it does:
- Reads a CSV of student marks (automation)
- Uses logistic regression to predict pass/fail

## Files:
- simple_automation_ml.py : Main code
- student_marks.csv : Sample data

## How to run:
1. Install pandas, sklearn: pip install pandas scikit-learn
2. Run: python simple_automation_ml.py
